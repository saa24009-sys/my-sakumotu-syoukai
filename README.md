# マイ作物 紹介サイト

このリポジトリは、「農業と情報」の実技テスト課題で作成した、きゅうりとナスの紹介Webサイトです。

## 💡 サイトの工夫した点（3つのポイント）
1. **スマホ対応（レスポンシブ）デザイン**：
   `meta viewport`タグを設定し、CSSで幅を％や最大幅（max-width）で指定することで、スマホから見ても文字や表、画像が崩れないように設計しました。
2. **ページ内リンクの設置**：
   作物を「きゅうり」と「ナス」の2つ紹介しているため、ページの上部に行きたい作物のセクションへ一発でジャンプできるナビゲーションリンクを作りました。
3. **JavaScriptによる動きの追加**：
   「栽培を応援する！」ボタンを設置し、クリックすると文字が「応援ありがとうございます！」に変わり、ボタンの色もオレンジからピンクに変化する動的な仕掛けを取り入れました。
